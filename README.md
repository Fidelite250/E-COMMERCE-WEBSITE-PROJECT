# Project Description
MJL TECHNOLOGY LTD is a modern e-commerce website developed using Shopify as a no-code platform. The website was created to provide customers with an easy and convenient online shopping experience for technology products and accessories.

The platform allows users to browse products, view product details, add items to cart, and contact the business directly through the contact page.
